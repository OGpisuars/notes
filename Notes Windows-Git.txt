- Git -
	Git is a software that comes with your pc if you have linux or macOS but if you have windows you have to download Git.
	Git is a memory card for code.
	Karoch kodeeshanas checkpoint

	git config --global user.name "[firstname lastname]" - iestāda username versijas vēsturei
	git config --global user.email "[valid email]" - uzstāda epasta adresi kura saistīta ar vēstures izveidotāju

	git --version - norāda kāda versija ir git un ir pieejams 


	mkdir gitVideo - uztaisa mapi/folder ar nosaukumu "gitVideo"
	cd -  Nomaina atrašanās vietu
	touch index.html app.css - izveido "index.html", "app.css"
	git init - lai izmantotu git. pašreizējo direktoriju pārtaisa par git repository initialize git
	git add  - saglabā specific lietu
	git add . -  saglabā visu
	git commit -m "message" - iesniedz savu darbu un lauj parādīt ko esi izdarījis. saglabā (commit) iepriekš sagatavotus failus git vēsturē.
	git log - saglabātie laiki
	git checkout "ievadītais save" - aiziet uz noteikto checkpoint/ branch
	git status - norāda failus darba directory kas ir sagatavoti saglabāšanai git vēsturē
	git add [file] - pievieno failu nākamai saglabāšanai  (stage array)
	git config --global user.name
